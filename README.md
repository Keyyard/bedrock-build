# @keyyard/bedrock-build

Compiler and build tooling for Minecraft Bedrock script add-ons. Bundles your TypeScript entry through esbuild, copies your behavior and resource packs into a `dist/` mirror, and provides `watch`, `deploy`, and `pack` workflows for fast iteration against the Bedrock retail client or any custom `com.mojang` directory.
